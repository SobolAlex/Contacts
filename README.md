# Contacts ASP.NET MVC Application

## Описание
ASP.NET Core MVC веб-приложение для управления контактами с полной поддержкой CRUD-операций.

Приложение позволяет:
- Просматривать список контактов
- Добавлять новые контакты
- Редактировать существующие контакты
- Удалять контакты
- Искать контакты по номеру телефона

Все данные хранятся в локальной базе данных SQLite.

## Архитектура
Решение построено с использованием принципов Clean Architecture и разделено на несколько проектов:

- **Contacts.Domain** — доменные сущности и бизнес-правила  
- **Contacts.Application** — DTO, запросы, интерфейсы  
- **Contacts.Infrastructure** — EF Core, репозитории, сервисы, работа с базой данных  
- **Contacts.Web** — ASP.NET Core MVC веб-приложение (UI)  

## Используемые технологии
- ASP.NET Core MVC  
- Entity Framework Core  
- SQLite  
- Bootstrap 5  
- JavaScript  
- Clean Architecture  
- Dependency Injection  

## Валидация
**Клиентская сторона (JavaScript + HTML):**
- Разрешён ввод только букв для полей «Имя» и «Должность»  
- Проверка формата номера телефона  
- Проверка обязательных полей  
- Минимальный возраст — 18 лет  

**Серверная сторона:**
- Валидация через DataAnnotations  
- Доменные проверки  
- Проверка уникальности номера телефона  

## База данных
SQLite. Файл базы данных создаётся автоматически с помощью `EnsureCreated()`.
